# UnityVertexMeshEditor

##### 유니티에서 런타임상에서 메쉬의 버텍스를 편집하는 간단한 프로젝트입니다.
##### 메쉬를 선택한 후 버텍스를 클릭하면 와이어프레임이 표시되고
##### 드래그 앤 드롭하여 움직 일 수 있습니다.
##### 여러개의 버텍스 다중선택이 가능합니다.
